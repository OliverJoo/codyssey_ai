## Codyssey E1-2 Mission

# 나만의 퀴즈 게임

## 프로젝트 개요
이 프로젝트는 Python 표준 라이브러리만 사용하여 구현한 터미널 기반 퀴즈 게임입니다.
사용자는 메뉴를 통해 퀴즈를 풀고, 새로운 퀴즈를 추가하며, 등록된 퀴즈 목록과 최고 점수를 확인할 수 있습니다.
퀴즈 데이터와 최고 점수는 `state.json` 파일에 저장되므로 프로그램을 다시 실행해도 이전 상태가 유지됩니다.
또한 저장 파일이 손상된 경우에는 예외 처리 후 기본 퀴즈 데이터로 초기화하고, 기존 파일은 `state.json.bak`로 백업하도록 구성했습니다.

---

## 퀴즈 주제 선정 이유
퀴즈 주제는 **Python 프로그래밍 기초**로 정했습니다.
이 주제를 선택한 이유는 Python 입문자가 자주 접하는 핵심 개념을 문제 형태로 반복 학습할 수 있기 때문입니다.
자료형, 리스트, 딕셔너리, 함수 정의, 주석 문법처럼 기초 문법에서 자주 헷갈리는 내용을 퀴즈로 구성하면 단순히 읽는 것보다 이해와 암기에 더 도움이 됩니다.
또한 프로젝트 자체도 Python으로 작성되어 있어 학습 주제와 구현 언어가 자연스럽게 연결된다는 점도 장점이라고 판단했습니다.

---

## 실행 방법
이 프로그램은 별도의 외부 라이브러리 설치 없이 Python 3 환경에서 실행할 수 있습니다.

```bash
python main.py
```

프로그램을 실행하면 메뉴가 출력되며, 번호를 입력해 원하는 기능을 선택할 수 있습니다.

---

## 기능 목록

| 번호 | 기능 | 설명 |
|:----:|------|------|
| 1 | 퀴즈 풀기 | 등록된 퀴즈를 순서대로 풀고, 정답 수를 기준으로 점수를 계산합니다. |
| 2 | 퀴즈 추가 | 문제, 선택지 4개, 정답 번호를 입력해 새 퀴즈를 등록합니다. |
| 3 | 퀴즈 목록 보기 | 현재 저장된 모든 퀴즈를 확인할 수 있으며, 정답은 `★` 표시로 구분됩니다. |
| 4 | 최고 점수 확인 | 지금까지 기록된 최고 점수를 확인합니다. |
| 5 | 종료 | 현재 상태를 저장한 뒤 프로그램을 종료합니다. |

추가로 입력값 검증 기능을 포함하여 빈 문자열, 숫자가 아닌 값, 범위를 벗어난 숫자를 입력했을 때 다시 입력하도록 처리했습니다.
또한 `Ctrl+C` 또는 입력 스트림 종료가 발생해도 현재 상태를 저장한 뒤 안전하게 종료하도록 구현했습니다.

## 데이터 파일 설명

### 1. `state.json`
프로그램의 현재 상태를 저장하는 JSON 파일입니다.
퀴즈 목록과 최고 점수가 함께 저장되며, 퀴즈 추가 시점, 최고 점수 갱신 시점, 프로그램 종료 시점에 파일이 갱신됩니다.

예시:
```json
{
    "quizzes": [
        {
            "question": "Python의 창시자는 누구인가요?",
            "choices": ["제임스 고슬링", "귀도 반 로섬", "리누스 토발즈", "데니스 리치"],
            "answer": 2
        }
    ],
    "high_score": 100
}
```

| 필드 | 타입 | 설명 |
|------|------|------|
| `quizzes` | list | 전체 퀴즈 목록 |
| `question` | str | 문제 내용 |
| `choices` | list[str] | 보기 4개 |
| `answer` | int | 정답 번호(1~4) |
| `high_score` | int | 최고 점수 |

### 2. `state.json.bak`
`state.json` 파일이 손상되었거나 형식이 올바르지 않을 때, 기존 파일 내용을 백업하기 위해 생성되는 파일입니다.
이후 프로그램은 기본 퀴즈 데이터로 초기화하여 계속 실행됩니다.
